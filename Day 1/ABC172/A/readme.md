[A-Calc](https://atcoder.jp/contests/abc172/tasks/abc172_a)

# Mô tả
Cho số nguyên a
in ra giá trị của a + a&sup2; + a&sup3;.  
  
# Điều kiện  
* 1 ≤ a ≤10
* *a* là số nguyên.

# Input
1 số nguyên dương a.

# Output
In ra giá trị của biểu thức a + a&sup2; + a&sup3;.

# Ví dụ
| Input | Output |
|-|-|
|2|14|
|10|1110|
