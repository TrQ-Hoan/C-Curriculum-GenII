[A-Calc](https://atcoder.jp/contests/abc172/tasks/abc172_a)

# Mô tả
Cho số nguyên a
in ra giá trị của a + a&sup2; + a&sup3.  
  
# Constraints  
* 1 ≤ a ≤10
* *a* là số nguyên.

# Input
Input is given from Standard Input in the following format:
```
a
```
# Output
Print the value a + a&sup2; + a&sup3; as an integer.

# Sample
| Input | Output |
|-|-|
|2|14|
|10|1110|
