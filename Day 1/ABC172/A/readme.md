# Mô tả
Given an integer a
as input, print the value a + a&sup2; + a&sup3;.  
  
# Constraints  
* 1 ≤ a ≤10
* *a* is an integer.

# Input
Input is given from Standard Input in the following format:
```
a
```
# Output
Print the value a + a&sup2; + a&sup3; as an integer.

# Sample
| Input | Output |
|-|-|
|2|14|
|10|1110|
