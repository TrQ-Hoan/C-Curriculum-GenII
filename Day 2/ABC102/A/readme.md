[A - Multiple of 2 and N](https://atcoder.jp/contests/ABC102/tasks/abc102_a)

# Mô tả
Cho số nguyên dương N. Hãy tìm số nguyên dương nhỏ nhất chia hết cho 2 và N.

# Điều kiện 
N là số nguyên.
1 ≤ N ≤ 10^9

# Input:
1 số nguyên N.

# Output
1 số thỏa mãn yêu cầu bài toán.

# Ví dụ
| Input | Output |
| ----- | ----- |
| 3 | 6 |
| 10 | 10 |
| 999999999 | 1999999998 |
