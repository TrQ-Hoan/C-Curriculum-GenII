# Mô tả
Cuộc thi "Lập trình cùng Fboiz" sẽ được tổ chức vào ngày hôm nay. Giám khảo của cuộc thi cho các thí sinh 2 số nguyên dương `N` và `D`. Nhiệm vụ của các thí sinh sẽ là tìm và in ra __số nhỏ thứ `N`__ trong những số **chia hết cho $100^`D`$**.

# Điều kiện
`D` chỉ có thể là 0, 1 hoặc 2.
`N` là số nguyên sao cho 1 ≤ `N` ≤ 100.

# Input
Input có dạng:
```
D N
```

# Output
1 số nguyên thỏa mãn.

# Example
|Input|Output|
|-|-|
|0 5|5|
