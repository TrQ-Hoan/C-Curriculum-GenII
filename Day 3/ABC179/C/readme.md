[C-AxB+C](https://atcoder.jp/contests/abc179/tasks/abc179_c)

# Mô tả
Cho một số nguyên dương $N$. Có bao nhiêu bộ 3 số nguyên dương ($A$,$B$,$C$) thỏa mãn $A$×$B$+$C$=$N$?

# Constraints
* 2 ≤ $N$ ≤ $10^{6}$
* $N$ là số nguyên.

# Input
Input có dạng:
```
N
```

# Output
In ra só bộ 3 thỏa mãn.

# Sample
|Input|Output|
|-|-|
|3|3|
|100|473|
|1000000|13969985|
