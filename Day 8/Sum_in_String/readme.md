Sum In String

# Mô tả
Cho xâu ký tự $S$ bao gồm các ký tự ‘a’,..,’z’ và các chữ số. 

Nhiệm vụ của bạn là hãy tính tổng các số có mặt trong xâu.

# Điều kiện
0 ≤ Length($S$) ≤ $10^{5}$

# Input
Input có dạng: 

$S$

# Output
Tổng các số xuất hiện trong xâu.

# Example
|Input|Output|
|-|-|
|1abc23|24|
|1abc2x30yz67|100|
